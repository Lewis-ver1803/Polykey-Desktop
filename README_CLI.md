# Usage
<!-- usage -->
```sh-session
$ npm install -g polykey-cli
$ polykey COMMAND
running command...
$ polykey (-v|--version|version)
polykey-cli/0.0.0 linux-x64 node-v8.11.4
$ polykey --help [COMMAND]
USAGE
  $ polykey COMMAND
...
```

## `polykey help [COMMAND]`

display help for polykey

```
USAGE
  $ polykey help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
